OONI Tor OK Rate
Success rate of Tor connections (0.0 = 0%, 1.0 = 100%)
About Tor reachability

Metric: fraction of Tor reachability measurements classified as OK in OONI data. Daily aggregation.

Use: drops signal active interference or breakage in the Tor bootstrap path. Cross-reference with HTTP dips and L3 spikes around events.

Source: OONI Aggregation API (tools: tor, snowflake, psiphon).
